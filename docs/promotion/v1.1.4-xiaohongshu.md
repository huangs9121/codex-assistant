# 小红书发布物料 · Codex Quota v1.1.4

## 标题

Codex 额度终于能常驻菜单栏了

## 正文

每次想看 Codex 还剩多少额度，都要手动输入 `/status`，真的很打断工作节奏。

更麻烦的是：用了重置卡、或者官方刚刚重置，页面里的额度有时还停在 1%，根本不知道到底有没有恢复。

所以我做了一个免费的 macOS 菜单栏小工具：**Codex Quota**。

现在抬眼看一眼菜单栏，就能知道：

✨ 当前还剩多少额度  
⏱️ 距离下次重置还有几天  
🔄 每 15 秒主动刷新真实额度  
🔔 检测到额度重置，第一时间发系统通知  
👀 时刻关注 Tibo 义父的 X，有 reset 预告第一时间知道，还能直接打开原帖  
🎨 4 种展示形式，Codex 文字 / OpenAI Logo / 隐藏都能自由切换  
📌 按住 Command 可以调整菜单栏位置  
🚀 支持开机启动，中英文跟随系统自动切换

隐私方面也尽量克制：不读取对话内容，不保存 Codex token。额度数据只从本机 Codex 获取，更新检查只访问公开 GitHub Release。

目前支持 Apple Silicon（M 系列芯片）和 macOS 13 及以上版本，项目免费开源。

下载地址：  
https://github.com/huangs9121/codex-assistant/releases/tag/v1.1.4

下载 `Codex.Quota-arm64.zip`，解压后拖进“应用程序”。如果首次打开被 macOS 拦截，可以在 Finder 里右键 App，再选择“打开”。

这是非官方开源项目，OpenAI Blossom 标识归 OpenAI 所有。
