## 一句话定位

不用反复输入 `/status`，在 Mac 菜单栏实时查看 Codex 剩余额度、重置倒计时和临时 reset 预告。

## 四个核心痛点

1. **频繁查额度打断工作**：想确认剩余额度时，要切回 Codex 再输入 `/status`。
2. **额度重置后旧数字容易误导**：本地旧快照还在，实际已经恢复，菜单栏却可能继续显示旧值。
3. **额度池容易读错**：Codex、Spark 等模型可能产生不同额度事件，简单取最新事件会显示成错误数字。
4. **重置时间不透明**：既要记常规周期，又要到 X 上寻找临时 reset 预告。

## 功能清单

- 每 15 秒自动读取本机 Codex 最新额度，无需手动刷新。
- 根据 macOS 首选语言自动显示简体中文或英文，其他语言回退英文。
- 只读取主 `codex` 周额度池，避免被 Spark 等独立模型额度覆盖。
- 到达额度周期重置时间时先恢复为 100%，新快照出现后以真实数据为准。
- 菜单栏可显示中文重置倒计时，例如 `6天` 或 `12小时`。
- 自动监控 Tibo reset 预告，发现可信消息后发送 macOS 通知。
- 有效预告可点击打开原始 X 帖子，过期后自动恢复“暂无预告”。
- 四种展示形式：原生电池、数字徽章、分段电池、纯数字。
- 三种标识形式：Codex 文字、OpenAI Logo、不显示标识。
- 显示精确到秒的更新时间、下次重置和当前套餐。
- 支持开机自动启动和 GitHub 新版本提醒。
- 按住 Command（⌘）拖动，可自由调整菜单栏位置。
- 原生 Swift App，开源免费，仅支持 Apple Silicon 和 macOS 13 及以上。

### 正文

我日常使用 Codex 时一直有几个小痛点：

- 想知道还剩多少额度，要切回 Codex 输入 `/status`；
- 额度已经重置，本地旧快照有时还会让工具显示旧数字；
- Codex 和其他模型的额度事件混在一起，简单读取“最新一条”可能读错；
- 临时 reset 通常要自己去 X 上找 Tibo 的预告。

所以做了一个免费的原生 macOS 菜单栏 App：**Codex Quota**。

它每 15 秒从本机自动读取 Codex 最新额度，菜单栏里直接显示剩余百分比和重置倒计时。它会区分主 Codex 周额度池，处理额度周期到点后的旧快照问题，还能监控 Tibo 的 reset 预告：有可信消息时发系统通知，点击即可打开原始 X 帖子，预告过期后自动清除。

为了不挤占菜单栏，它提供原生电池、数字徽章、分段电池、纯数字四种样式；左侧可以显示 Codex、OpenAI Logo，也可以完全隐藏。按住 Command（⌘）拖动还能调整菜单栏位置。

其他功能包括：

- 精确到秒的更新时间；
- 中文下次重置时间；
- 当前套餐；
- 开机自动启动；
- GitHub 新版本提醒。

项目开源免费，数据主要从 `~/.codex/sessions` 本机只读获取，不上传 Codex token、账号信息或对话内容。网络只用于检查 GitHub 公开版本和只读获取 reset 预告。

目前支持 Apple Silicon 和 macOS 13 以上。安装包尚未经过 Apple 公证，首次打开需要在 Finder 中右键 App，选择“打开”。

GitHub：<https://github.com/huangs9121/codex-assistant>

如果你也经常用 Codex，欢迎试一下。我最想确认两个问题：额度数字在你的账号上是否始终准确，以及你更喜欢哪一种菜单栏样式。

## Product Hunt 英文草稿

### Name

Codex Quota

### Tagline

See your Codex quota and reset countdown in the Mac menu bar

### Description

Codex Quota is a free, open-source macOS menu bar app for Codex users who are tired of repeatedly running `/status`.

It refreshes the remaining quota every 15 seconds from local Codex snapshots, isolates the primary Codex weekly quota from model-specific pools, handles stale snapshots after a scheduled reset, and shows the next reset in a compact Chinese countdown.

It can also monitor Tibo reset announcements, send a native macOS notification, and link directly to the original X post. Four display styles and three identity modes let users keep the menu bar as compact as they want.

The app is built with native Swift, is free and open source, and supports Apple Silicon Macs running macOS 13 or later.
