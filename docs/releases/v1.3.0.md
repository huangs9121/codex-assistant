# Codex Quota v1.3.0

本版本增加 Tibo 的 Codex 额度重置预告监控：

- 每 5 分钟读取结构化预告数据；来源服务器约每 30 分钟刷新 Tibo 的公开帖子。
- 只处理“可能重置”“已预告”和“已发起”三类明确信号，不对普通 `reset` 关键词盲目通知。
- 新的 reset 相关消息通过 macOS 通知推送，同一条消息只通知一次。
- 点击通知可打开 Tibo 的 X 原帖。
- 应用菜单显示当前重置预告状态和预期时间；无法可靠解析时间时显示“时间待确认”。

当前发布使用 ad-hoc 签名且未经 Apple 公证，因此 Gatekeeper 出现安全提醒属于预期情况。
