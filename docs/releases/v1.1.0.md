# Codex Quota v1.1.0

Codex Quota 是非官方的开源 macOS 菜单栏工具。本版本包括：

- 左侧标识支持 Codex 文字、OpenAI Logo、隐藏三种状态。
- 可选在菜单栏显示紧凑重置倒计时（`D` / `H`）。
- 信息区新增当前套餐和套餐到期；Pro 套餐来自本地额度快照，到期时间无法安全匹配时显示 `--`。
- 支持 GitHub 新版本提醒；仅提示并打开 Release 页面，仍需手动下载安装。
- 支持开机启动，额度数据继续每 15 秒从本机刷新。
- 仅支持 arm64（Apple Silicon）和 macOS 13 及以上。

## 安装

1. 下载 `Codex Quota-arm64.zip` 并解压。
2. 将 `Codex Quota.app` 拖入 `/Applications`。
3. 首次运行如果被 macOS 拦截，可在 Finder 中右键 App 后选择“打开”。

当前发布使用 ad-hoc 签名且未经 Apple 公证，因此 Gatekeeper 出现安全提醒属于预期情况。请仅从本仓库的 GitHub Releases 下载。
