# Codex Quota v1.1.3

本版本汇总 v1.1.2 之后的开发成果：

- 展示形式和标识形式改为二级菜单，新增带百分号的纯数字样式。
- 菜单栏的重置倒计时改用中文“天 / 小时”显示。
- 额度只读取主 `codex` 周额度池，不再被 Spark 等独立模型额度覆盖。
- 本地快照声明的重置时间到达后，额度立即按 100% 显示；新快照到来后仍以真实值为准。
- 新增 Tibo reset 预告监控和 macOS 通知，忽略低置信信号并避免重复通知。
- 预告到达预期时间后自动恢复“暂无预告”；有效预告可点击打开 Tibo 的 X 原帖。
- 移除无法稳定获取的“套餐到期”信息。

当前发布使用 ad-hoc 签名且未经 Apple 公证，因此 Gatekeeper 出现安全提醒属于预期情况。
